Healthcare Claims, Provider & Data Quality Analytics
Project Overview

Developed a healthcare analytics solution using SQL Server to analyze member, provider, and claims data across a healthcare insurance environment. The project focused on evaluating claims processing performance, provider network utilization, healthcare spending trends, and data quality metrics to support operational decision-making and data governance initiatives.
